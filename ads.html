Password security tips

Start with length

Use at least 16 characters and keep every character type enabled for maximum randomness.

  • Increase length further for critical accounts (20+ characters).
  • Avoid predictable phrases or keyboard patterns.

Keep it unique

Every site should get its own password so a single breach cannot unlock other accounts.

  • Let a reputable password manager store and fill the credentials.
  • Add MFA wherever offered for an extra layer of defense.

Refresh safely

Rotate passwords for sensitive services, especially after a security alert or device loss.

  • Revoke access for old sessions when you update credentials.
  • Back up recovery codes offline in case you lose a device.
